  cat-baskets.jpg   →  Any basket photo
  cat-single.jpg    →  A single flower/stem photo
  cat-custom.jpg    →  A custom/special order photo

ABOUT:
  about.jpg      →  A photo of you crafting / your workspace

HOW TO SAVE FROM INSTAGRAM:
  1. Open each post on Instagram
  2. Tap the 3 dots (...) → "Copy Link" 
     OR right-click the image → "Open image in new tab"
  3. Save the image and rename it as above
  4. Drop it into this folder

Once all photos are added, just refresh the browser and all images will appear!
